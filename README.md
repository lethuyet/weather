# Weather APP


## Local setup
### Prerequisite
1. Node.js
2. Yarn
3. Git

### Setup using Terminal
```
cd {YOUR CUSTOM DIRECTORY}
git clone git@github.com:lethuyet/weather
cd weather
yarn && cd client && yarn
```
### Launch local servers to serve React Client and Node Server apps
At "weather" root:
```
yarn local
```
